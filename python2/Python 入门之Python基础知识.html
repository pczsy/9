    Python 入门之Python基础知识

    1、变量

    （1）变量就是把程序运行的中间结果临时存在内存中,以便后续代码使用

    （2）变量的作用:

    昵称,就是代指内存中某个地址中的内容

    a = 123

    变量名 赋值 值

    （3）变量命名的规则：

    (1)、 变量名由字母、数字、下划线组成

    (2)、 变量名不能以数字开头

    (3)、 变量名要具有可描述性

    (4)、 变量名要区分大小写

    (5)、 变量名禁止使用Python关键字

    (6)、 变量名不能使用中文和拼音（不是不能使用，太low）

    (7)、 变量名推荐写法：

            ``` 驼峰体：AgeOfOldboy```

         ```下划线:（推荐使用）：age_of_oldboy```
