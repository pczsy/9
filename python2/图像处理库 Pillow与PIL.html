    PIL只支持python2的版本到2.7；

    Python imaging Library ；

    Pillow 是PIL派生的一个分支，支持3以上Python版本。

    命令使用pip安装：

    pip install Pillow

    简单示例使用：

    from PIL import Image

    img = Image.open("example.jpg")

    img.rotate(45).show()

    结论：将图片顺时针旋转45度展示。

    内部调用系统本地默认图片展示软件来展示图像。
