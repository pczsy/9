    　　字符串

    　　　　定义方法：

    • 单引号，name = 'lalaxing'
    • 双引号，name = "lalaxing"
    • 三引号，name = '''lalaxing'''

    　区别如下：

    >>>sentence = "I'm fine."    # 内部可以加单引号
>>> sentence1 = 'I\'m fine.' # 可加转义符
>>> print (sentence,sentence1)
I'm fine. I'm fine.
>>> paragraph = '''I'm fine.
thank you.'''                 #可换行，表示段落
>>> print (paragraph)
I'm fine.
thank you.

    　　　　字符串拼接：　　　　

    >>> s1 = '床前明月光，'
>>> s2 = '遗是地上霜。'
>>> s1 + s2
'床前明月光，遗是地上霜。'
>>> s1*4
'床前明月光，床前明月光，床前明月光，床前明月光，'
