    python 数据类型 常用法方

    upper() 大写 str
    lower() 小写 str
    strip() rstrip() lstrip() 去除字符两边的空格 去右边 左边空白 str
    replace() 将指定的字符替换成想替换的 str
    split() 将字符串按指定的方式进行分割，返回的是列表 str
    isdigit () 判断是否为数字 str
    startswith() 判断是否以指定的字符开头(返回布尔值，True 或者False) str
    endswitch() 判断是否以指定的字符结尾(返回布尔值，True 或者False) str
    format() 表现形式{索引}，可以通过索引，对指定字符进行赋值 str
    encode() 程序默认是用Unicode编码进行保存，可以将字符串的保存编码通过encode进行修改 str
    join() 按照指定得连接方式，拼接字符串,不是字符串拼接不了 str
    append() 直接添加要追加的元素内容，并会在列表元素的最后面追加 list
    insert() insert(指定索引，"添加的元素")，在指定的索引之前追加要插入的元素 list
    reverse() 把列表进行旋转 list
    sort() 对列表进行排序 list
    remove() 这里直接填写要删除的元素，而不是索引" list
    pop() 直接指定要删除的元素的索引 list
    extend() 循环列表的每个元素（元组没有此功能） list
    clear() 清空列表中所有的元素 list
    del() 指定要删除的元素索引 list
    keys() 取出所有的key值 list
    values() 取出所有的value值 list
    items() 取出字典中的keys和values list
    get() 通过get取key或者value list
    update() 把字典的键/值对更新到dict里，不存在，则添加,存在，则更新 list/set
    add() 为集合添加元素 set
    discard() 删除集合中指定的元素 set
    intersection() 取集合的交集 set
    union() 返回两个集合的并集 set
    difference() 返回多个集合的差集 set
    symmetric_difference() 返回两个集合中不重复的元素集合，即会移除两个集合中都存在的元素。 set
