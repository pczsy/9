    我正在寻找的是可以将“明天早上6点”或“中午的下一个模拟”转换为适当的日期时间对象。

     

    解决方案

    parsedatetime - 能够解析“人类可读”日期/时间表达式的Python模块。

    #!/usr/bin/env python
from datetime import datetime
import parsedatetime as pdt # $ pip install parsedatetime

cal = pdt.Calendar()
now = datetime.now()
print("now: %s" % now)
for time_string in ["tomorrow at 6am", "next moday at noon", 
                    "2 min ago", "3 weeks ago", "1 month ago"]:
   print("%s:\t%s" % (time_string, cal.parseDT(time_string, now)[0]))
