    • Windows 下安装 Python + Django
    原文出处:本文由博客园博主IEnumer提供。
    原文连接:https://www.cnblogs.com/wxmayifei/p/11324313.html

    Django是Python的一个Web开发框架，以下是介绍的是windows下的安装步骤，

    作者的环境是Win10 ，Windows Server 也是一样的

    以下是作者整理的步骤，也可以参考官方教程

    （1）安装Python ，这个直接从官网下载安装就行了，安装最新版本的就行了

     

    （2）安装pip

             打开命令行运行以下命令，安装最新版pip

    curl https://bootstrap.pypa.io/get-pip.py -o get-pip.py
python get-pip.py

    　　pip更新命令

    python -m pip install -U pip

     

    （3）安装virtualenv 

    　　打开命令行运行以下命令

    python -m pip install virtualenvwrapper-win

     

    （4）为即将开发的站点建立一个工作目录，假设项目名称叫test

    　　在命令行中进入到该目录，然后运行以下命令，建立虚拟环境test

    python -m virtualenv test

         创建完虚拟环境后需要在同一目录下运行以下命令激活

    test\Scripts\Active

    　　注意，这儿的test名称和上面的虚拟环境名称是相同的

    　　停止运行虚拟环境可以在激活后运行以下命令

    test\Scripts\Deactive

     

    （5）安装Django，在上面激活的虚拟环境下运行下面的命令

    python -m pip install django

    　　创建项目，以下命令行是创建一个叫test_project的项目（别漏掉最后一个点）

    django-admin.py startproject test_project .

     

    （6）测试是否可正常是使用

    python manage.py runserver

    　　打开浏览器，输入以下地址，看看是否能正常访问

    http://127.0.0.1:8000/

    　　默认是8000端口， 如果已被占用，以下是指定端口号的方式

    python manage.py runserver 8001

    　

    （7）创建供Django使用的数据库（SQLite）

    python manage.py migrate

    　　可以看到目录中多了一个db.sqlite3文件

     

    完成！！！
