    第一课

    Python的优缺点以及干什么用的，想必大家都很了解。主要有：简单易学，功能强大，可移植性强等，是人工智能、运维领域必备编程语言。

    1.安装，在Python官网下载最新版本，在安装时点击环境变量设置复选框，这样不用手动设置环境变量。

    安装好后，在开始菜单中点击IDLE，进入编译器。至此所需要准备的环境及工具准备完善，正是进入Python编程。

    2.第一个程序就是我们著名的“hello world！”

    print（‘hello world！’），3.0版本以后必须要加括号，否则会出错。Python有严格的缩进格式要求，每个缩进为4个空格，整个程序不能tab和空格混种用，必须前后一致，否则会出错。

    3.变量不需要申明，不需要删除，可以直接回收适用。type()查询数据类型。主要的数据类型有：整数，浮点数，真值，字符串。

    4.tuple和lis两种序列，前者元素不可变，后者可变。字符串是tuple。

    5.运算，数字运算，判断和逻辑（真假）。

    6.选择的语句 选择用if :elif: elif: else: 格式要求：所有的if语句后面都要有冒号，同时隶属关系用4个空格缩进。包括后面的函数定义都要用冒号。

    7.循环语句 for while continue break

    8.函数 

    def function_name(a,b,c):
    statement
    return something

    9.将东西归为类，类下面细分子类，子类继承父类的属性；引用对象的属性object.attribute。

    10.通过self调用类属性；__init__()在建立对象时自动执行；类属性和对象的性质的区别。

    11.dir()用来查询一个类或者对象的所有属性；help（）查询说明文档。
