    2.函数的使用

    　　直接使用函数名()调用函数块。

    1 def say_hello():
2     print("Hello")
3     
4 
5 say_hello()

    3.函数的参数

    在函数名后面的小括号内部填写 参数

    def 函数名(参数1, 参数2, 参数3，……):

    　　函数封装的代码

    　　……

    1 def sum_2_num(num1, num2):
2     """两数求和"""
3     print(num1 + num2)
4 
5 
6 sum_2_num(333, 555)

     

    4.形参和实参

    • 形参：定义函数是，小括号中的参数，是用来不接受参数用的，在函数内部作为变量使用
    • 实参：调用函数时，小括号中的参数，是用来吧数据传递到函数内部用的

    注：上面的代码中：num1,num2是形参；333，555是实参

    5.函数的返回值

    在函数中使用return关键字返回结果

    1 def sum_2_num(num1, num2):
2     """两数求和"""
3     return num1 + num2
4 
5 
6 sum1 = sum_2_num(333, 555)
7 print(sum1)

     

    注：return作为函数的结束，其后代码无效。
